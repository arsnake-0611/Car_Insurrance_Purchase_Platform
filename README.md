# Car Insurance Purchase Platform

A web-based platform for purchasing car insurance policies, including quotation, vehicle information management, and a simple insurance dashboard. The project was originally developed as an EA project for the Hong Kong Institute of Vocational Education.

## Features

- User sign-in and basic account handling (see `signin` folder).
- Car insurance quotation pages and logic (see `quotation` folder).
- Vehicle information pages and styles (see `vehicle` folder).
- Simple insurance dashboard and vehicle sales handling modules (`insuranceDashboard`, `vehicleSalesHandling`).
- Frontend built with HTML, CSS, and JavaScript (approx. 36.8% JS, 32.3% CSS, 30.9% HTML).

## Project Structure

Key folders and files in this repository:[page:0]

- `.vscode` – Editor configuration for Visual Studio Code.
- `icon/` – Icons and image assets used in the UI.
- `insuranceDashboard/` – Pages and scripts for the insurance dashboard.
- `quotation/` – Quotation-related pages, styles, and scripts.
- `signin/` – Sign-in and login related pages.
- `temp/` – Temporary or experimental pages and files.
- `vehicle/` – Vehicle information and related UI components.
- `vehicleSalesHandling/` – Vehicle sales handling and related logic.
- `option.html`, `option.css`, `option.js` – Main option/landing page and its styles and scripts.
- `info.css` – Global or shared styling for information pages.
- `USERS.txt`, `errorInput.txt` – Sample user data and error-input examples for testing.
- `Hong Kong Institute of Vocational Education.docx`, `Project AY2425v2.docx` – Project guideline and documentation files.
