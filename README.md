# Car Insurance Purchase Platform

A web-based platform for purchasing car insurance policies, including quotation, vehicle information management, and a simple insurance dashboard. The project was originally developed as an EA project for the Hong Kong Institute of Vocational Education.[page:0]

## Features

- User sign-in and basic account handling (see `signin` folder).[page:0]
- Car insurance quotation pages and logic (see `quotation` folder).[page:0]
- Vehicle information pages and styles (see `vehicle` folder).[page:0]
- Simple insurance dashboard and vehicle sales handling modules (`insuranceDashboard`, `vehicleSalesHandling`).[page:0]
- Frontend built with HTML, CSS, and JavaScript (approx. 36.8% JS, 32.3% CSS, 30.9% HTML).[page:0]

## Project Structure

Key folders and files in this repository:[page:0]

- `.vscode` – Editor configuration for Visual Studio Code.[page:0]
- `icon/` – Icons and image assets used in the UI.[page:0]
- `insuranceDashboard/` – Pages and scripts for the insurance dashboard.[page:0]
- `quotation/` – Quotation-related pages, styles, and scripts.[page:0]
- `signin/` – Sign-in and login related pages.[page:0]
- `temp/` – Temporary or experimental pages and files.[page:0]
- `vehicle/` – Vehicle information and related UI components.[page:0]
- `vehicleSalesHandling/` – Vehicle sales handling and related logic.[page:0]
- `option.html`, `option.css`, `option.js` – Main option/landing page and its styles and scripts.[page:0]
- `info.css` – Global or shared styling for information pages.[page:0]
- `USERS.txt`, `errorInput.txt` – Sample user data and error-input examples for testing.[page:0]
- `Hong Kong Institute of Vocational Education.docx`, `Project AY2425v2.docx` – Project guideline and documentation files.[page:0]
